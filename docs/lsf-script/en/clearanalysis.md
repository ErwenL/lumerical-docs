# clearanalysis

Clears analysis object results. This data is also cleared by switching from Analysis
Mode to Layout Mode.

Note: The analysis object results are calculated with the runanalysis command.

| **Syntax**                             | **Description**                                                         |
| -------------------------------------- | ----------------------------------------------------------------------- |
| clearanalysis;                         | Clears analysis object results. This function does not return any data. |
| clearanalysis( "name1", "name2", ...); | Clears data from specific analysis objects.                             |
