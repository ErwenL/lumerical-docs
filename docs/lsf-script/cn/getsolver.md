# getsolver

返回 该 求解器 该 是 currently active. 

注意  : 'getsolver' 是 deprecated, use 'getactivesolver' instead   
---  
**语法** |  **描述**  
---|---  
?getsolver;  |  返回 该 求解器 该 是 currently active.   
  
**示例**

If EME 是 active, 那么 getsolver 将 show it: 
    
    
    ?getsolver;
    警告: prompt line 1: 'getsolver' 是 deprecated, use 'getactivesolver' instead
    EME
