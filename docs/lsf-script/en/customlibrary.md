# customlibrary

Returns the location (path) of the Custom library.

| **Syntax**           | **Description**                              |
| -------------------- | -------------------------------------------- |
| out = customlibrary; | Returns the directory of the custom library. |

**Example**

```
?out = customlibrary;
>C:/Users
```
