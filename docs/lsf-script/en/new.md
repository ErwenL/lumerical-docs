# new

Same as newproject. It creates a new simulation project file. If there is an existing
project file in the GUI, the old project file will not be saved.

| **Syntax** | **Description**             |
| ---------- | --------------------------- |
| new;       | Creates a new project file. |

**Examples**

Creates a new project with the newcommand.

```
new;
```
