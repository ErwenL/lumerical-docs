# upper

转换 all 该 characters 在 一个 字符串 到 upper case. 

**语法** |  **描述**  
---|---  
upper(字符串);  |  转换 一个 字符串 到 upper case.   
  
**示例**

Use upper case 用于 all 该 characters 在 一个 字符串. 
    
    
    ?upper("name");
    NAME
