# getvariable

返回 该 值 的 一个 workspace 变量, 如果 该 值 是 not found it 返回 一个 ‘default’ 值. 

**语法** |  **描述**  
---|---  
out= getvariable (属性,default);  |  返回 该 值 的 一个 workspace 变量, 如果 该 值 是 not found it 返回 一个 ‘default’ 值.   
  
**示例**
    
    
    ?getvariable("变量");
