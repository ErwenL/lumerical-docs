# unique

返回矩阵中所有唯一值的数组。换句话说，此命令返回矩阵中的所有元素，如果它们在矩阵中出现多次则不会重复。

**语法** | **说明**
---|---
out=unique(a); | 返回包含矩阵a中所有唯一值的数组。

  * 此命令主要用于整数值。
  * 它也适用于浮点数，但这取决于软件内部使用的精度，因此在两个数字差异很小的情况下可能会失败（见下面的示例）。


**示例**

定义一个矩阵并在脚本提示窗口中显示所有唯一元素。


    A = [1,2;2,3];  
    ?unique(A);  
    result:   
    1   
    2   
    3   
      
    format long; # 将精度设置为16位数字  
    A= [2., 2.000000000000001; 3, 4.5];  
    ?unique(A);  
    result:   
    2.000000000000000   
    2.000000000000001   
    3.000000000000000   
    4.500000000000000   
      
    format short; # 将精度设置为8位数字  
    B= [2., 2.0000000000000001; 3, 4.5];  
    ?unique(B);  
    result:   
    2   
    3   
    4.5
