# matlabget

将变量从 MATLAB 工作区复制到脚本变量工作区。结果变量将与 MATLAB 变量具有相同的名称，并将覆盖任何具有相同名称的现有变量。如果变量在 MATLAB 中不存在，该命令将返回错误。更多信息，请参阅 matlab 命令描述。

注意：要使用此命令，必须启用 Matlab 脚本集成。关于如何设置此功能的更多信息，请参阅 Matlab 脚本集成页面。

**语法** | **描述**
---|---
matlabget(var1, var2,...varN); | 此命令的参数是一个或多个引用 MATLAB 工作区中变量的变量名。此函数不返回任何数据。
