# print

Prints a string.

| **Syntax**            | **Description**                                  |
| --------------------- | ------------------------------------------------ |
| print("hello world"); | Prints the string defined in the quotation mark. |

**Examples**

```
a = "my string";
print(a);my string
print('Hello World!');Hello World!
```
