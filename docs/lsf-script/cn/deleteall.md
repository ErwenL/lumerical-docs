# deleteall

Deletes all 对象 在 该 current group scope. 

**语法** |  **描述**  
---|---  
deleteall; |  Deletes all 对象 在 该 current group scope. This 函数 does not 返回 any 数据.  
  
**示例**

This 函数 是 often used 在 creating 一个 group, eg., [Structure Groups](/hc/en-us/articles/360034382434-Structure-Groups). 

**参见**

[Manipulating 对象](/hc/en-us/articles/360037228834), [groupscope](/hc/en-us/articles/360034928553-groupscope)
