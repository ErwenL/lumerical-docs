# log10

计算 该 base 10 logarithm 函数. Input 可以 为 complex 或 negative. 

**语法** |  **描述**  
---|---  
out = log10(x);  |  The base 10 logarithm 的 x. Input 可以 为 complex 或 negative.   
  
**示例**

计算 该 base 10 logarithm 的 一个 数字. 
    
    
    ?log10(100);
     result: 
     2
