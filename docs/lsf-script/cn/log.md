# log

计算自然对数函数。输入可以是复数或负数。

**语法** | **描述**
---|---
out = log(x); | x的自然对数。输入可以是复数或负数。

**示例**

检查e的自然对数是否为1：

```lsf
?Exp=exp(1);
result: 
2.71828 
?log(Exp); # 结果应该是1
result: 
1  
```
