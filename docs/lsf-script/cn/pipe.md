# |（或）- 脚本运算符

是逻辑或函数。忽略 x 和 y 的虚部。

**语法** | **描述**
---|---
out = y | x; | 如果 x、y 的实部中任一或两者非零，则返回 1。否则返回 0。
y or x; | 与 | 相同。

**示例**

此示例演示 "|" 和 OR 函数的用法。

```
? (2) | (4);
result:
1
?(3 > 4) or (4 >3);
result:
1
? (0+1i) | (0);
result:
0
```
