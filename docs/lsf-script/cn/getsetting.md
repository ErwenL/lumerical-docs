# getsetting

返回 该 值 的 一个 用户定义 setting. This 命令 可以 为 used 使用 [ setsetting ](/hc/en-us/articles/360034927973-setsetting) . 

**语法** |  **描述**  
---|---  
getsetting("name");  |  获取 该 值 的 一个 用户定义 setting.   
  
**Parameter** |  **Type** |  **描述**  
---|---|---  
name  |  字符串  |  name 的 该 setting.   
  
**示例**

Please visit 该 page [ setsetting ](/hc/en-us/articles/360034927973-setsetting) .
