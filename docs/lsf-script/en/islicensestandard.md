# islicensestandard

Checks if the license is a standard license.

| **Syntax**         | **Description**                                                                                                                      |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| islicensestandard; | Checks if the current license for Lumerical is a standard license. It returns true if it is a standard license, and false otherwise. |

**Example**

```
#Ran on an enterprise license environment  
?islicensestandard;  
  
Result:  
False
```
