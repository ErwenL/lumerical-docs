# removeexpansion

从模式展开监视器中删除 DFT 监视器。

**语法** | **描述**
---|---
removeexpansion("name"); | 从所选模式展开监视器的 "Monitors for expansion" 列表中删除指定名称的 DFT 监视器。

**示例**

请参阅 [setexpansion](./setexpansion.md) 命令。

**另请参见**

- [addmodeexpansion](./addmodeexpansion.md)
- [setexpansion](./setexpansion.md)
