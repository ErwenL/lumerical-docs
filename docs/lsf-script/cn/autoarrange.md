# autoarrange

**语法** | **描述**
---|---
autoarrange (name); | Arrange port positions and 维度 of compound or scripted 元素 automatically. This 命令 does not 返回 anything.

**示例**

If a compound named "COMPOUND_1" is created,using this 命令
     autoarrange("COMPOUND_1");
will arrange the positions.

If a compound named "COMPOUND_1" is created,using this 命令
     autoarrange("COMPOUND_1");
will arrange the positions.
