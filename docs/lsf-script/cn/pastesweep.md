# pastesweep

将扫描/优化/蒙特卡罗分析项从剪贴板粘贴到"优化和扫描"选项卡。

**语法** | **描述**
---|---
pastesweep("name"); | 将扫描/优化/蒙特卡罗分析项从剪贴板粘贴。"name" 是父项的绝对名称，新的分析项将作为子项粘贴到此父项下。如果名称为空，则将新的分析项粘贴为最顶层的项。返回新项的绝对名称。如果粘贴失败则返回空字符串。

有关实现示例，请参阅 copysweep 脚本页面。
