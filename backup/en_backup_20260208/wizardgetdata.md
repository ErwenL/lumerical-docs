# wizardgetdata

Returns data entered into a specific widget. 

**Syntax** |  **Description**  
---|---  
out = wizardgetdata(N);  |  Returns the value that the user entered into the Nth widget. Out will be a number or a string, depending on the type of widget.

See the newwizard page for an example.
