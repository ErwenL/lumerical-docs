# tan

Calculates the trigonometric tangent function. Angle units are in radians. The function
is defined for complex angles. Phase of a complex number is evaluated between - π and π
.

| **Syntax**    | **Description**                   |
| ------------- | --------------------------------- |
| out = tan(x); | Returns the complex tangent of x. |

**Example**

Calculate tan( π /4 + i).

```
theta=pi/4+1i;
?tan(theta);
result: 
0.265802+0.964028i 
```
