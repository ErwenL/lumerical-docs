# setdevice

设置用于运行仿真的设备类型。此命令仅当存在 FDTD 仿真区域时才能使用。

**语法** | **描述**
---|----
setdevice("device_type"); | 通过 device_type 设置用于运行仿真的设备类型：

- device_type：设备类型，可以是 "CPU" 或 "GPU"
