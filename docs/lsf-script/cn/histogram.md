# histogram

创建包含产量分析结果直方图计数的矩阵。

**语法** |  **描述**
---|---
out = histogram(y);  |  返回包含 y 的直方图计数的矩阵。
out = histogram(y,n);  |  返回包含 y 的直方图计数的矩阵，使用 n 个 bin。
