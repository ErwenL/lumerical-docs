# runwizard

Runs 该 wizard 和 返回 一个 值 indicating 该 button was pressed. 

**语法** |  **描述**  
---|---  
out = runwizard;  |  返回 either 0, +1 或 -1.  0 means 该 用户 pressed Cancel, 1 means 该 用户 pressed 该 first button (created 通过 calling newwizardpage) 和 -1 means 该 用户 pressed 该 second button (created 通过 calling newwizardpage).   
  
**示例**

See 该 newwizard page 用于 一个 example.
