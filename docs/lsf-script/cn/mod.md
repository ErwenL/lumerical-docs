# mod

求除法后的模。

**语法** | **描述**
---|---
out = mod(x,y); | 返回 x - n*y，其中 n = floor(x/y)（如果 y 不等于 0）。输入 x 可以是实数数组或实数标量，而 y 必须是实数标量。按照惯例，以下为真：mod(x,0) = x mod(x,x) = 0

**示例**

求矩阵中每个元素除以标量后的模。

```
?A=[1,2;3,4];
?mod(A,2);
result:
1 2
3 4
result:
1 0
1 0
```
