# createbeam

**语法** | **描述**
---|---
createbeam; | Creates a Gaussian beam in the deck/global workspace.  返回 the name of the Gaussian beam created, which is by default "gaussian#" (# being the total 数字 of Gaussian beams already existing in the current deck + 1).
out = createbeam; | Creates a Gaussian beam in the deck/global workspace and saves its name in the 变量 "out".

**示例**

The following 脚本 命令 will create a Gaussian beam in the deck and print its name in the 脚本 prompt. 
    ?createbeam;

The following 脚本 命令 will create a Gaussian beam in the deck and print its name in the 脚本 prompt. 
    ?createbeam;
