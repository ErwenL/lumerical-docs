# randnmatrix

Initialize a matrix. All elements are normally distributed random numbers with mean 0
and standard distribution 1.

| **Syntax**                   | **Description**                                                                                                                        |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| x = randnmatrix(i,j,k,....); | Initializes an i x j x k x .... matrix. The elements are all random normally distributed numbers with mean 0 and standard deviation 1. |
