# asterisk

[FDTD](/hc/en-us/articles/360033154434) [模式](/hc/en-us/articles/360020687354) [DGTD](/hc/en-us/articles/360037744173) [CHARGE](/hc/en-us/articles/360037184494) [HEAT](/hc/en-us/articles/360037224694) [FEEM](/hc/en-us/articles/360037744633) [INTERCONNECT](/hc/en-us/articles/360037304774)
Performs a multiplication operation. When applied to matrices, this 运算符 does simple 元素 by 元素 multiplication. For 矩阵 multiplication, see the [ mult ](/hc/en-us/articles/360034925813-mult) 命令.

**语法** | **描述**
---|---
z = x * y; | Multiply x and y.

**示例**

This 示例 shows the usage of the "*" 运算符.
    x=1:3;
    y=2:4;
    ?z=x*y;
    result: 
    2  
    6  
    12
