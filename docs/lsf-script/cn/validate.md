# validate

Updates 该 results 用于 该 specified analyzer.

**语法** |  **描述**  
---|---  
validate("analyzer"); |  Updates 该 results 用于 该 specified "analyzer". If 该 name 是 not provided, 该 选中的 analyzer 将 为 updated.  
  
**示例**
    
    
    addelement("Optical Network Analyzer");  
    validate("ONA_1");
