# pwd

Returns the current working directory. 

**Syntax** |  **Description**  
---|---  
out = pwd;  |  Returns the current working directory as a string.   
  
**Examples**

Prints the current working directory to the screen. 
    
    
    ?path=pwd;
    C:\demo
