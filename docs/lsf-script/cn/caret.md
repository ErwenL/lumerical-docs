# ^

Indicates 该 power. In expression A^B, 如果 B 是 complex, 该 phase 的 A 是 evaluated 从 -  π  到  π  . 

**语法** |  **描述**  
---|---  
y = x^3;  |  x cubed.   
  
**示例**

This example shows 该 usage 的 该 "^" operator. 
    
    
    x=1:3;
    y=2:4;
    ?z=x^y;
    result: 
    1  
    8  
    81
