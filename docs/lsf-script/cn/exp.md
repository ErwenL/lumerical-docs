# exp

计算自然指数函数。输入可以是复数。

**语法** | **描述**
---|---
out = exp(x); | x 的自然指数。

**示例**

计算 e，使用 exp(1)；

```powershell
?exp(1);
结果：
2.71828
```

将 1+1i 乘以 30 度。

```powershell
x=1+1i;
?x * exp(1i*pi/6);
结果：
0.366025+1.36603i
```
