# newwizardpage

This creates a page for the wizard and should be done before adding any widgets. 

**Syntax** |  **Description**  
---|---  
newwizardpage( "label1");  |  Creates a button with the label "label1". Typically, this will be "Done" or "OK".   
newwizardpage( "label1", "label2");  |  Creates two buttons with labels "label1" and "label2". These will typically be "Next" and "Back" or "Done" and "Back".
