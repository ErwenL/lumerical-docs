# loadsweep

将扫描对象与先前生成的扫描结果一起加载。

**语法** |  **描述**
---|---
loadsweep;  |  将先前生成的扫描结果加载到模拟中的所有扫描对象。
loadsweep("name");  |  将先前生成的扫描结果加载到模拟中指定的扫描对象。

**示例**

    loadsweep;
