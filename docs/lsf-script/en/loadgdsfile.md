# loadgdsfile

Loads a gds file to a layer builder object. The command only functions properly when a layer builder object is included in the simulation and is selected.

**Syntax** |  **Description**  
---|---  
loadgdsfile("gds_example.gds"); |  Loads the gds file named "gds_example" to the layer builder object.
