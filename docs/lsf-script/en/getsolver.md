# getsolver

Returns the solver that is currently active. 

Note  : 'getsolver' is deprecated, use 'getactivesolver' instead   
---  
**Syntax** |  **Description**  
---|---  
?getsolver;  |  Returns the solver that is currently active.   
  
**Example**

If EME is active, then getsolver will show it: 
    
    
    ?getsolver;
    Warning: prompt line 1: 'getsolver' is deprecated, use 'getactivesolver' instead
    EME
