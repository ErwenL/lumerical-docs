# ceil

Rounds 该 input 到 该 nearest integer greater than 或 equal 到 itself. 

**语法** |  **描述**  
---|---  
out = ceil(x);  |  返回 该 nearest integer greater than 或 equal 到 x.   
  
**示例**

Find 该 nearest integer greater than 或 equal 到 1.41: 
    
    
    ?ceil(1.41);
    result:
    2
