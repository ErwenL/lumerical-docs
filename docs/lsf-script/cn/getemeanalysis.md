# getemeanalysis

    ?getemeanalysis("group spans");

**语法** | **描述**
---|---
?getemeanalysis; | Lists all the parameters in the analysis window.
getemeanalysis("property"); | Gets the 值 of the calculation 参数 named "property" in EME 求解器 analysis window.

**示例**

This code will display the 值 of the "group spans" parameters from the EME setup section of the EME analysis window.
