# getanalysis

Lists 该 参数 used 在 MODE' FDE 和 Finite Element IDE 分析 window.

**语法** |  **描述**  
---|---  
?getanalysis; |  Lists all 该 参数 在 该 分析 window.  
getanalysis("属性"); |  返回 该 current 值 用于 该 particular 属性 在 该 分析 window
