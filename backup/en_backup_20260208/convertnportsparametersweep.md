# convertnportsparametersweep

Converts the textual S-Parameter sweep file (.txt file) to a binary file (.idat file).

**Syntax** |  **Description**  
---|---  
convertnportsparametersweep("filename") |  Converts the textual S-Parameter sweep file "filename" to a binary file (.idat file).  
**Example** The following script converts the S-Parameter sweep file "s_para_sweep.txt" to a binary file "s_para_sweep.idat".
    
    
    convertnportsparametersweep("s_para_sweep.txt")
