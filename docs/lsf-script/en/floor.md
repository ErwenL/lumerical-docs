# floor

Rounds the input to the nearest integer less than or equal to itself.

| **Syntax**      | **Description**                                      |
| --------------- | ---------------------------------------------------- |
| out = floor(X); | Returns the nearest integer less than or equal to X. |

**Example**

Find the nearest integer greater than or equal to 1.41:
