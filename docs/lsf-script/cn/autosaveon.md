# autosaveon

This 命令 turns 在 该 feature 到 automatically save 该 current project before running 一个 仿真.

注意 该 此 only applies 到 simulations, not sweeps. Running 一个 sweep 将 always save 该 project 文件 before 该 sweep 是 run, regardless 的 该 autosave setting.

**语法** |  **描述**  
---|---  
autosaveon; |  Automatically saves current project before running 一个 仿真.
