# randnmatrix

初始化一个矩阵。所有元素都是均值为 0、标准差为 1 的正态分布随机数。

**语法** | **描述**
---|---
x = randnmatrix(i,j,k,....); | 初始化一个 i x j x k x .... 的矩阵。所有元素都是均值为 0、标准差为 1 的正态分布随机数。
