# gdsclose

This function closes a GDSII file for writing. Before calling this command, a .gds file has to be previously opened, see [gdsopen](/hc/en-us/articles/360034927093-gdsopen).

**Syntax** |  **Description**  
---|---  
gdsclose("filename") |  closes a .gds file in the current working directory.  
**Parameter** |  **Type** |  **Description**  
---|---|---  
filename |  string |  name of the GDSII file to export, may also contain a file path.
