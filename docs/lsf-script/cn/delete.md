# delete

Deletes 选中的 对象. 

**语法** |  **描述**  
---|---  
delete;  |  Deletes 选中的 对象.  This 函数 does not 返回 any 数据.   
  
**示例**

创建 一个 对象 和 那么 delete it, just 用于 illustration. 
    
    
    addrect;
    设置("name","substrate");
    select("substrate");
    delete;
