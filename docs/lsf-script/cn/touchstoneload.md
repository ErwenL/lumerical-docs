# touchstoneload

从包含 Touchstone 文件格式 S 参数的文件中加载无源网络数据。有关 Touchstone 规范的更多信息，请参阅[此页面](https://helpfiles.keysight.com/csg/N1930xB/FilePrint/SnP_File_Format.htm)。

**语法** | **描述**
---| ---
out = touchstoneload (filename); | 返回一个矩阵，其中第一列包含以 Hz 为单位的频率值。S 参数使用 MA 格式返回，其中 M 是幅度，A 是弧度角。

### 示例

在这个简单的示例中，加载了一个 s2p 格式的 Touchstone 文件。

```lsf
#filename PA3-110.s2p
out=touchstoneload('PA3-110.s2p');
#display loaded values
?out
result:
```
