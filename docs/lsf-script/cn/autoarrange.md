# autoarrange

The 脚本 命令 arranges 端口 positions 和 dimensions 的 compound 或 scripted elements automatically. Equivalently it 可以 also 为 done 通过 pressing ‘Arrange’ 在 该 元素 端口 editor tab.

**语法** |  **描述**  
---|---  
autoarrange (name); |  Arrange 端口 positions 和 dimensions 的 compound 或 scripted elements automatically. This 命令 does not 返回 anything.  
  
**示例**

If 一个 compound named "COMPOUND_1" 是 created,使用 此 命令
    
    
     autoarrange("COMPOUND_1");

将 arrange 该 positions.
