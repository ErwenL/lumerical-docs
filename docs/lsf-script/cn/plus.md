# +（加号）- 脚本运算符

执行加法运算。

**语法** | **描述**
---|---
y = x + z; | 将 x 和 z 相加。
y = string1 + string2; | 将字符串连接在一起。

**示例**

此示例演示 "+" 运算符的用法。

```
x=1:3;
y=2:4;
?z=x+y;
result:
3
5
7
?"file_"+"name_"+num2str(1);
file_name_1
```
