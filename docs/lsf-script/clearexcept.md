# clearexcept

Clears the workspace variables except the specified ones. 

**Syntax** |  **Description**  
---|---  
clearexcept(a);  |  Clears the workspace variables except for variable a.   
  
**Examples**
    
    
    a = "my string";
    b = 1;
    d = 2;
    clearexcept(a);
