# exp

Calculates the natural exponential function. Input can be complex.

| **Syntax**    | **Description**               |
| ------------- | ----------------------------- |
| out = exp(x); | The natural exponential of x. |

**Example**

To calculate e, use exp(1);

```
?exp(1);
result: 
2.71828 
```

To multiply 1+1i by 30 degrees

```
x=1+1i;
?x * exp(1i*pi/6);
result: 
0.366025+1.36603i 
```
