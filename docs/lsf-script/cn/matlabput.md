# matlabput

将变量从 Lumerical 脚本工作区复制到 MATLAB 工作区。MATLAB 工作区中的结果变量将与 Lumerical 中的变量同名，并将覆盖任何具有相同名称的现有变量。如果变量在 Lumerical 工作区中不存在，该命令将返回错误。

更多信息，请参阅 matlab 命令描述。

**语法** | **描述**
---|---
matlabput(var1, var2,...varN); | 此命令的参数是一个或多个存在于 Lumerical 变量工作区中的变量名。此函数不返回任何数据。

请参阅 matlab 函数描述中的示例。
