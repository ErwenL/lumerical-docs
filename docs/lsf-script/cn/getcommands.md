# getcommands

返回 该 list 的 available 脚本 commands 在 该 current 脚本 workspace. 

**语法** |  **描述**  
---|---  
?getcommands;  |  返回 一个 list 的 available 脚本 commands   
  
**示例**

返回 该 list 的 available 脚本 commands 在 该 current 脚本 workspace. 
    
    
    ?getcommands;
    +
    -
    *
    /
    ^
    ==
    !=
    =
