# cross

**语法** | **描述**
---|---
C = cross(A, B); | 返回 the cross product of A and B

**示例**

计算 the cross product of (1,1,0) and (0,1,0).
    A=[1,1,0];
    B=[0,1,0];
    ?C=cross(A,B);
    返回: 
    0  0  1

计算 the cross product of (1,1,0) and (0,1,0).
    A=[1,1,0];
    B=[0,1,0];
    ?C=cross(A,B);
    返回: 
    0  0  1
