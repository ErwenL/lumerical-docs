# loadgdsfile

将 gds 文件加载到层构建器对象。仅当模拟中包含层构建器对象并将其选中时，该命令才能正常工作。

**语法** |  **描述**
---|---
loadgdsfile("gds_example.gds"); |  将名为 "gds_example" 的 gds 文件加载到层构建器对象。
