# / (除法，正斜杠) - 脚本运算符

执行除法运算。此运算符执行简单的逐元素除法。

**语法** | **描述**
---|---
y = x / z; | x除以z。

**示例**

此示例展示了"/"运算符的用法。

```
x=1:3;
y=2:4;
?z=x/y;
结果：
0.5 
0.666667 
0.75 
```
