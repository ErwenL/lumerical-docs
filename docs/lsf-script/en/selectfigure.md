# selectfigure

Selects a figure, which will be shown on screen (gives it focus). A warning will be
generated if the figure does not exist.

| **Syntax**       | **Description**                                                                   |
| ---------------- | --------------------------------------------------------------------------------- |
| selectfigure;    | Selects the last figure that was created. This function does not return any data. |
| selectfigure(1); | Selects figure 1.                                                                 |

**Example**

See exportfigure example.
