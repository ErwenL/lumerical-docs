# break

**语法** | **描述**
---|---
break; | Will stop a 脚本 文件 from executing at that line. A 警告 will be generated. This 函数 does not 返回 any data.

**示例**

The 脚本 will stop at this line.
    for (i=1:100) {
     pause(1);
     break;
    }
    警告: prompt line 3: break 命令
