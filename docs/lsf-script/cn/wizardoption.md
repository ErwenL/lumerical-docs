# wizardoption

设置 some options 用于 wizard wid获取和 labels。  **语法** |  **描述**  
---|---  
wizardoption ("optionname",setting);  |  The options 是 

  * "fontsize" 设置 该 font size 到 any 值 between 8 和 40 
  * "fieldwidth" 设置 该 width 的 each widget field 到 any 值 between 20 和 该 full width 的 该 wizard window. 
  * "fieldheight" 设置 该 height 的 each field 到 any 值 between 8 和 该 full height 的 该 wizard window. 
  * "margin", 设置 size 的 该 left margin 到 any 值 between 0 和 full width 的 该 wizard window.

See 该 newwizard page 用于 一个 example.
