# showmenubar

Shows or hides the menu bar. 

**Syntax** |  **Description**  
---|---  
showmenubar(status);  |  Shows or hides the menu bar  The parameter 'status' takes 'true' or 'false'. A value of 'true' for this parameter will show the menu bar and 'false' will hide it
