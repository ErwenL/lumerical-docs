# chebin

**语法** | **描述**
---|---
chebin(f,x,xi,xmin,xmax) | Interpolates the 函数 f onto the xi points. It assumes that f contains the samples of the 函数 taken on the Chebyshev roots grid specified in x; x must be constructed by the call # x = chpts(xmin,xmax,NumPts), otherwise an error is returned.

See 示例 for the chpts 命令.

See 示例 for the chpts 命令.
