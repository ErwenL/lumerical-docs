# uninstalldesignkit

Uninstalls a design kit file in the Design Kits folder.

| **Syntax**                               | **Description**                                                                                                                              |
| ---------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| uninstalldesignkit("name", ["version"]); | Uninstalls a design kit named 'name' with version 'version' in the Design Kits folder. "version" is optional and default is an empty string. |

**Example**

```
# uninstalls a design kit folder "LCML" with version v1.1 from the Design Kits folder
uninstalldesignkit("LCML", "v1.1");
```
