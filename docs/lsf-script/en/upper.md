# upper

Converts all the characters in a string to upper case.

| **Syntax**     | **Description**                  |
| -------------- | -------------------------------- |
| upper(string); | Converts a string to upper case. |

**Example**

Use upper case for all the characters in a string.

```
?upper("name");
NAME
```
