# simulationdiverged

In layout mode, check whether the simulation reached the divergence checking auto
shutoff threshold.

| **Syntax**              | **Description**                                                                  |
| ----------------------- | -------------------------------------------------------------------------------- |
| out=simulationdiverged; | Returns 1 if the simulation reached the auto shutoff max threshold, 0 otherwise. |
