# iscell

The script command checks whether input is a cell.

| **Syntax**            | **Description**                                                                                                       |
| --------------------- | --------------------------------------------------------------------------------------------------------------------- |
| value= iscell(input); | Determine whether ‘input is a cell. It returns logical 1 (true) if ‘input’ is a cell and logical 0 (false) otherwise. |

### Example

```
>x=cell(3);
>?iscell(x);
result: 
1 
```
