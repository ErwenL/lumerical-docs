# getemeanalysis

获取 calculation 参数 在 MODE' EME 分析 window. 

**语法** |  **描述**  
---|---  
?getemeanalysis;  |  Lists all 该 参数 在 该 分析 window.   
getemeanalysis("属性");  |  获取 该 值 的 该 calculation 参数 named "属性" 在 EME 求解器 分析 window.   
  
**示例**

This code 将 display 该 值 的 该 "group spans" 参数 从 该 EME setup section 的 该 EME 分析 window. 
    
    
    # display 该 值 的 该 group spans  
    ?getemeanalysis("group spans");
