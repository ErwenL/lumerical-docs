# getpath

**语法** | **描述**
---|---
out = getpath; | 返回 the current 路径 as a 字符串. Use ?getpath; to print it to the screen.

**示例**

Gets the current 路径.
    ?getpath;
    ./  
    C:/Program Files/Lumerical/2020a/scripts

Gets the current 路径.
    ?getpath;
    ./  
    C:/Program Files/Lumerical/2020a/scripts
