# add2visualizer

向现有可视化工具添加数据。

**语法** |  **描述**  
---|---  
add2visualizer( dataset, visualizer number )  |  此命令向现有可视化工具添加数据。如果没有与可视化工具编号对应的可视化工具，则忽略该命令。   
   
**示例**

参见visualize命令的示例。
