# runinitialize

此脚本命令用于初始化逐步仿真。

**语法** | **描述**  
---|---  
runinitialize; | 初始化逐步仿真，与 'run' 不同的是，此命令仅对仿真进行初始化。此命令应与 [runstep](runstep.md) 和 [runfinalize](runfinalize.md) 配合使用，通常用于联合仿真。
