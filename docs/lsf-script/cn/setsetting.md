# setsetting

> **原文**: setsetting  
> **翻译日期**: 2026-02-03  
> **翻译状态**: ✅ 已完成

设置用户自定义设置的值。用户设置会被永久保存，即使关闭应用程序后仍然可用。

| **语法** | **说明** |
| --- | --- |
| setsetting("name","string_value") | 设置用户自定义设置的值。 |

| **参数** | **类型** | **说明** |
| --- | --- | --- |
| name | 字符串 | 设置的名称。 |
| string_value | 字符串 | 设置的值。 |

**示例**

此命令允许用户定义自定义设置，该设置将被永久存储，直到软件重新安装。

```
setsetting("in","out");
?getsetting("in");
out # 设置"in"的"string_value"
```
