# getcommands

Returns the list of available script commands in the current script workspace.

| **Syntax**    | **Description**                             |
| ------------- | ------------------------------------------- |
| ?getcommands; | Returns a list of available script commands |

**Examples**

Returns the list of available script commands in the current script workspace.

```
?getcommands;
+
-
*
/
^
==
!=
=
```
