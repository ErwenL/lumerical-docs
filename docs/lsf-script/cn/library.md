# library

返回当前安装的元素库中可用的元素列表，包括自定义元素。

**语法** |  **描述**
---|---
out = library;  |  返回当前安装的元素库中可用的元素列表，包括自定义元素。

**示例**

列出当前可用的元素

    ?library
    result:
    ::analyzers::digital::logic analyzer
    ::analyzers::electrical::oscilloscope
    ...
