# zeros

Initialize 一个 矩阵. All elements 是 设置 到 zero (same as [ 矩阵 ](/hc/en-us/articles/360034929613-矩阵) ). 

**语法** |  **描述**  
---|---  
x = zeros(i,j,k,....);  |  Initializes 一个 i x j x k x .... 矩阵.   
  
**示例**
    
    
    ?x=zeros(2,2,2);
    result(i,j,1):
    0 0 
    0 0 
    result(i,j,2):
    0 0 
    0 0
