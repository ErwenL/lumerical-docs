# clearexcept

**语法** | **描述**
---|---
clearexcept(a); | Clears the workspace variables except for 变量 a.

**示例**

    a = "my 字符串";
    b = 1;
    d = 2;
    clearexcept(a);

    a = "my 字符串";
    b = 1;
    d = 2;
    clearexcept(a);
