# getrectangle

**语法** | **描述**
---|---
out=getrectangle ("element",”w”); | 返回 the width of an 元素 rectangle.
out=getrectangle ("element",”h”); | 返回 the height of an 元素 rectangle.

**示例**

To get the x position of a waveguide 元素 named "Straight Waveguide_1" use the following 脚本 
    ?getrectangle("Straight Waveguide_1","w") ;
    result: 
    1e-006
