# saveelement

Saves an element to a .ice file in the current working directory. 

**Syntax** |  **Description**  
---|---  
saveelement ("name");  |  Save an element to a file. The element will be saved with the current element name in the current working directory with extension ICE.   
  
**Example**
    
    
    #adds an element star coupler and saves it to an .ice file in the current working directory
    addelement("Star Coupler");
    saveelement("STAR_1");
