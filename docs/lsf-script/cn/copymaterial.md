# copymaterial

Makes 一个 copy 的 一个 材料 在 该 材料 database. 

**语法** |  **描述**  
---|---  
out = copymaterial("materialname");  |  创建 一个 copy 的 该 材料 "materialname". The 新的 name 是 returned.
