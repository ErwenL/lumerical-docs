# loadsweep

Loads the sweep object with the previously generated sweep result.

| **Syntax**         | **Description**                                                                       |
| ------------------ | ------------------------------------------------------------------------------------- |
| loadsweep;         | Loads previously generated sweep result to all the sweep objects in simulation.       |
| loadsweep("name"); | Loads previously generated sweep result to the specified sweep objects in simulation. |

**Example**

```
loadsweep;
```
