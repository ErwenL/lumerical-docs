# &

Is the logical AND function. Imaginary components of x and y are ignored.

| **Syntax**   | **Description**                                                                        |
| ------------ | -------------------------------------------------------------------------------------- |
| out = y & x; | If the real part of either or both of x,y are zero, then return 0. Otherwise return 1. |
| y and x;     | Same as &.                                                                             |

**Examples**

This example shows the usage of the "&" and AND function.

```
? (2) & (4);
result:
1
?(3  4) and (4 3);
result: 
0 
? (0+1i) & (1);
result:
0
```
