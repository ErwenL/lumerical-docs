# convertnportsparametersweep

**语法** | **描述**
---|---
convertnportsparametersweep("filename") | Converts the textual S-参数 sweep 文件 "文件名" to a binary 文件 (.idat 文件).

**示例**

    convertnportsparametersweep("s_para_sweep.txt")

    convertnportsparametersweep("s_para_sweep.txt")
