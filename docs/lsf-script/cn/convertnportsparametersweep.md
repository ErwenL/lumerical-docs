# convertnportsparametersweep

转换 该 textual S-Parameter sweep 文件 (.txt 文件) 到 一个 binary 文件 (.idat 文件).

**语法** |  **描述**  
---|---  
convertnportsparametersweep("文件名") |  转换 该 textual S-Parameter sweep 文件 "文件名" 到 一个 binary 文件 (.idat 文件).  
**示例** The following 脚本 转换 该 S-Parameter sweep 文件 "s_para_sweep.txt" 到 一个 binary 文件 "s_para_sweep.idat".
    
    
    convertnportsparametersweep("s_para_sweep.txt")
