# randmatrix

Initialize a matrix. All elements are random numbers between 0 and 1.

| **Syntax**                  | **Description**                                                                              |
| --------------------------- | -------------------------------------------------------------------------------------------- |
| x = randmatrix(i,j,k,....); | Initializes an i x j x k x .... matrix. The elements are all random numbers between 0 and 1. |

**Examples**

```

0.570605 0.89404
```
