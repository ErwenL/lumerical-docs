# deletesweep

删除指定的参数扫描、优化或蒙特卡洛分析任务。

**Syntax** |  **Description**  
---|---  
deletesweep("name"); |  删除具有指定名称的扫描、优化或蒙特卡洛分析任务。  
  
 **示例**

 此示例展示如何删除名为"thickness_sweep"的任务。请从[参数扫描](/hc/en-us/articles/360034922873-Parameter-sweeps)页面下载示例文件。
    
    
    deletesweep("thickness_sweep");
