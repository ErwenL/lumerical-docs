# clearfunctions

Clears all 或 specified stored functions. 

**语法** |  **描述**  
---|---  
clearfunctions;  |  Clears all workspace functions.  This 函数 does not 返回 any 数据.   
clearfunctions(func1, func2, ...);  |  Clears only 该 workspace functions 使用 该 specified names.
