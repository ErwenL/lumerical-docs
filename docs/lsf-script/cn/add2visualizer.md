# add2visualizer

向现有可视化器添加数据。

**Syntax** | **Description**
---|---
add2visualizer( dataset, visualizer number ) | 此命令向现有可视化器添加数据。如果没有与可视化器编号对应的可视化器，则忽略该命令。

**Example**

参见visualize命令的示例。
