# pwd

返回当前工作目录。 

**Syntax** |  **Description**  
---|---  
out = pwd;  |  以字符串形式返回当前工作目录。   
  
**Examples**

将当前工作目录打印到屏幕。 
    
    
    ?path=pwd;
    C:\demo
