# getanalysis

Lists the parameters used in MODE' FDE and Finite Element IDE analysis window.

| **Syntax**               | **Description**                                                              |
| ------------------------ | ---------------------------------------------------------------------------- |
| ?getanalysis;            | Lists all the parameters in the analysis window.                             |
| getanalysis("property"); | Returns the current value for the particular property on the analysis window |
