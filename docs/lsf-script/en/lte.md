# <=

Performs a logical less-than-or-equal-to comparison. Imaginary components of x and y are ignored. 

**Syntax** |  **Description**  
---|---  
out = y <= x;  |  Less than or equal to.   
  
**Examples**

This example shows the usage of the "<=" operator. 
    
    
    x=[1+3i, 2+5i];
    y=[0.5+4i, 3+10i];
    ?x<=y;
    Warning: prompt line 3: in expression A <= B, imaginary parts of A and B are ignored
    result: 
    0  1
