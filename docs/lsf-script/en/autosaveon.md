# autosaveon

This command turns on the feature to automatically save the current project before
running a simulation.

Note that this only applies to simulations, not sweeps. Running a sweep will always save
the project file before the sweep is run, regardless of the autosave setting.

| **Syntax**  | **Description**                                                  |
| ----------- | ---------------------------------------------------------------- |
| autosaveon; | Automatically saves current project before running a simulation. |
