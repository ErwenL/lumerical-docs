# clearanalysis

Clears 分析 对象 results. This 数据 是 also cleared 通过 switching 从 Analysis Mode 到 Layout Mode. 

注意: The 分析 对象 results 是 calculated 使用 该 runanalysis 命令. 

**语法** |  **描述**  
---|---  
clearanalysis;  |  Clears 分析 对象 results.  This 函数 does not 返回 any 数据.   
clearanalysis( "name1", "name2", ...);  |  Clears 数据 从 specific 分析 对象.
