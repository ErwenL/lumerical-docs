# matrix

Initialize a matrix. All elements are set to zero.

| **Syntax**              | **Description**                         |
| ----------------------- | --------------------------------------- |
| x = matrix(i,j,k,....); | Initializes an i x j x k x .... matrix. |

**Examples**

```
?x=matrix(2,2,2);
result(i,j,1):
0 0 
0 0 
result(i,j,2):
0 0 
0 0 
```
