# materialexists

Returns a boolean indicating whether or not a material exists in the material database.

| **Syntax**                      | **Description**                                                                                         |
| ------------------------------- | ------------------------------------------------------------------------------------------------------- |
| materialexists("materialname"); | Returns 1 if the material named "materialname" is present in the material database. Otherwise return 0. |
