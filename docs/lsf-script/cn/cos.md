# cos

计算 该 trigonometric cosine 函数. Angle units 是 在 radians. The 函数 是 defined 用于 complex angles. Phase 的 一个 complex 数字 是 evaluated between -  π  和  π  . 

**语法** |  **描述**  
---|---  
out = cos(x);  |  返回 该 complex cosine 的 x.   
  
**示例**

计算 cos(  π  /4 + i). 
    
    
    theta=pi/4+1i;
    ?cos(theta);
    result: 
    1.09112-0.830993i
