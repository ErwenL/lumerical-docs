# help

使用默认 Web 浏览器打开 Lumerical 知识库。

**语法** |  **描述**
---|---
help(argument="");  |  使用默认 Web 浏览器打开 Lumerical 知识库。如果未提供参数，Web 浏览器将打开包含所有脚本命令字母顺序列表的页面；否则将使用 'argument' 参数运行搜索，并打开搜索结果页面。

**示例**

    >help;
    >help("setsparameter");
