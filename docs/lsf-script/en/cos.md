# cos

Calculates the trigonometric cosine function. Angle units are in radians. The function
is defined for complex angles. Phase of a complex number is evaluated between - π and π
.

| **Syntax**    | **Description**                  |
| ------------- | -------------------------------- |
| out = cos(x); | Returns the complex cosine of x. |

**Example**

Calculate cos( π /4 + i).

```
theta=pi/4+1i;
?cos(theta);
result: 
1.09112-0.830993i 
```
