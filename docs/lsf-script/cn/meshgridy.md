# meshgridy

在 y 方向创建 2D 网格。

**语法** | **描述**
---|---
out = meshgridy(x,y); | 如果 x 和 y 分别是维度为 nX1 和 mX1 的单列（或单行）向量，则命令

  * Y = meshgridy(x,y);

将创建维度为 nXm 的 2D 矩阵，其中 Y(i,j)=y(j)。

**示例**

参阅 meshgridx 中的示例。
