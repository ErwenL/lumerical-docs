# meshgridy

Create a 2D meshgrid in the y direction 

**Syntax** |  **Description**  
---|---  
out = meshgridy(x,y);  |  If x and y are single column (or single row vectors), of dimension nX1 and mX1 respectively, the command 

  * Y = meshgridy(x,y); 

Will create a 2D matrix of dimension nXm where Y(i,j)=y(j).
