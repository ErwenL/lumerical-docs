# wizardgetdata

返回 数据 entered into 一个 specific widget. 

**语法** |  **描述**  
---|---  
out = wizardgetdata(N);  |  返回 该 值 该 该 用户 entered into 该 Nth widget. Out 将 为 一个 数字 或 一个 字符串, depending 在 该 类型 的 widget.   
  
**示例**

See 该 newwizard page 用于 一个 example.
