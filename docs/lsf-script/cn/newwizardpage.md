# newwizardpage

这会为向导创建一个页面，应在添加任何构件之前完成。

**语法** | **描述**
---|---
newwizardpage( "label1"); | 创建标签为 "label1" 的按钮。通常这会是 "Done" 或 "OK"。
newwizardpage( "label1", "label2"); | 创建两个标签为 "label1" 和 "label2" 的按钮。通常这会是 "Next" 和 "Back" 或 "Done" 和 "Back"。
