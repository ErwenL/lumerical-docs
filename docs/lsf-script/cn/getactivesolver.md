# getactivesolver

**语法** | **描述**
---|---
?getactivesolver; | List the active 求解器.

**示例**

When "EME" 求解器 is already added, the following 脚本 will give the result: 
    setactivesolver("EME");
    ?getactivesolver;
    EME

When "EME" 求解器 is already added, the following 脚本 will give the result: 
    setactivesolver("EME");
    ?getactivesolver;
    EME
