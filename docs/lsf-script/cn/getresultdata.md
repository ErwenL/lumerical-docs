# getresultdata

**语法** | **描述**
---|---
?getresultdata; | 返回 the names of all 元素 in the current 仿真 that contain results.
?getresultdata("analyzer"); | 返回 all available results for "analyzer".
out = getresultdata("analyzer", "result"); | 返回 the result "result" for "analyzer".
