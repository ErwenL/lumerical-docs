# clearexcept

Clears 该 workspace variables except 该 specified ones. 

**语法** |  **描述**  
---|---  
clearexcept(一个);  |  Clears 该 workspace variables except 用于 变量 一个.   
  
**示例**
    
    
    一个 = "my 字符串";
    b = 1;
    d = 2;
    clearexcept(一个);
