# lower

Converts all the characters in a string to lower case.

| **Syntax**     | **Description**                  |
| -------------- | -------------------------------- |
| lower(string); | Converts a string to lower case. |

**Example**

Use lower case for all the characters in a string.

```
?lower("NAME");
name
```
