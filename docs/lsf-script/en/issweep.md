# issweep

The script command checks if the simulation is in sweep mode.

| **Syntax**     | **Description**                                                     |
| -------------- | ------------------------------------------------------------------- |
| out = issweep; | Returns true (1 or 0) if the simulation is currently in sweep mode. |
