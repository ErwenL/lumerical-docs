# getvariable

**语法** | **描述**
---|---
out= getvariable (property,default); | 返回 the 值 of a workspace 变量, if the 值 is not found it 返回 a ‘default’ 值.

**示例**

    ?getvariable("变量");

    ?getvariable("变量");
