# meshgrid3dx

Create a 3D meshgrid in the x direction

| **Syntax**                | **Description**                            |
| ------------------------- | ------------------------------------------ |
| out = meshgrid3dx(x,y,z); | The 3D version of meshgridx and meshgridy. |
