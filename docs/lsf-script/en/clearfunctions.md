# clearfunctions

Clears all or specified stored functions.

| **Syntax**                         | **Description**                                                         |
| ---------------------------------- | ----------------------------------------------------------------------- |
| clearfunctions;                    | Clears all workspace functions. This function does not return any data. |
| clearfunctions(func1, func2, ...); | Clears only the workspace functions with the specified names.           |
