# asapimport

**语法** | **描述**
---|---
asapimport( "sourcename"); | Imports the fld 文件 into the sourcename 光源. A 文件 browser will open to select the 文件.  This 函数 does not 返回 any data.
asapimport( "sourcename", "filename"); | Specify the 文件 to open.
