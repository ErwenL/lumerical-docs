# savesweep

Creates and saves the simulations files to a folder in the working directory, this script command is equivalent to the GUI "Save to files" operation. 

**Syntax** |  **Description**  
---|---  
savesweep;  |  Saves the simulation files of all sweep objects in simulation.   
savesweep("name");  |  Saves the simulation files of the specified sweep object.   
**Example**
    
    
    savesweep;
