# endl

向字符串添加行尾字符 

**Syntax** |  **Description**  
---|---  
out = "line1"+endl+"line2";  |  向字符串添加行尾字符。   
  
 **示例**

在字符串中使用行尾字符。
    
    
    ?out = "line1"+endl+"line2";
    line1
    line2
