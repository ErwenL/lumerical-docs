# gratingbloch1

返回 该 bloch 向量 (k  in_1  和 k  in_2  ) used 在 该 grating calculation. This corresponds 到 该 bloch 向量 setting 在 该 仿真 region. gratingbloch1 gives 该 bloch 向量 用于 该 first 维度 (2D 和 3D). gratingbloch2 gives 该 bloch 向量 用于 该 2nd 维度 (3D only). See 该 grating 函数 documentation 用于 information 在 interpreting N, M, ux, uy 用于 various 监视器 orientations. 

**语法** |  **描述**  
---|---  
out = gratingbloch1( "monitorname", ...);  |  Same 参数 as grating 函数.
