# tan

计算三角正切函数。角度单位为弧度。该函数对复数角度也有定义。复数的相位在-π和π之间评估。

**语法** | **描述**
---|---
`out = tan(x);` | 返回x的复数正切值。

**示例**

计算tan(π/4 + i)。

```
theta=pi/4+1i;
?tan(theta);
result: 
0.265802+0.964028i
```
