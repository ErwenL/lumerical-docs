# isstruct

The script command checks whether input is a structure. 

**Syntax** |  **Description**  
---|---  
value= isstruct(input);  |  Determine whether ‘input’ is a structure. It returns logical 1 (true) if ‘input’ is a structure and logical 0 (false) otherwise.   
  
###  Example 
    
    
    >x=struct;
    >?isstruct(x);
    result: 
    1
