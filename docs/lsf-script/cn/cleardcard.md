# cleardcard

Clears global d-cards. Only global d-cards 是 cleared. Local d-cards 是 associated 使用 该 current 仿真 和 可以 only 为 cleared 通过 switching 从 Analysis Mode 到 Layout Mode. 

**语法** |  **描述**  
---|---  
cleardcard;  |  Clears all 该 global d-cards.  This 函数 does not 返回 any 数据.   
cleardcard( "name1", "name2", ...);  |  Clears any 数字 的 specified d-cards.
