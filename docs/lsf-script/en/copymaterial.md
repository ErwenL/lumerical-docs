# copymaterial

Makes a copy of a material in the material database.

| **Syntax**                          | **Description**                                                          |
| ----------------------------------- | ------------------------------------------------------------------------ |
| out = copymaterial("materialname"); | Creates a copy of the material "materialname". The new name is returned. |
