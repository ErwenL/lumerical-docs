# getpath

获取 该 current path. By default, 该 current working directory 和 该 脚本 sub-directory 的 该 installation directory 是 在 该 path. Typically 该 path 是 important 用于 setting 该 location 的 your 脚本 files. 

**语法** |  **描述**  
---|---  
out = getpath; |  返回 该 current path as 一个 字符串. Use ?getpath; 到 print it 到 该 screen.  
  
**示例**

获取 该 current path.
    
    
    ?getpath;
    ./  
    C:/Program Files/Lumerical/2020a/scripts
