# +

Performs an addition operation. 

**Syntax** |  **Description**  
---|---  
y = x + z;  |  Add x and z.   
y = string1 + string2;  |  Concatenate strings together.   
  
**Examples**

This example shows the usage of the "+" operator. 
    
    
    x=1:3;
    y=2:4;
    ?z=x+y;
    result: 
    3  
    5  
    7  
    ?"file_"+"name_"+num2str(1);
    file_name_1
