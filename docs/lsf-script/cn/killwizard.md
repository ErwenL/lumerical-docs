# killwizard

这将关闭向导窗口。仅在向导窗口已使用 newwizard 命令创建后，才应调用此命令。

**语法** |  **描述**
---|---
killwizard;  |  这将关闭向导窗口。

**示例**

有关示例，请参阅 newwizard 页面。
