# log

计算 该 natural logarithm 函数. Input 可以 为 complex 或 negative. 

**语法** |  **描述**  
---|---  
out = log(x);  |  The natural logarithm 的 x. Input 可以 为 complex 或 negative.   
  
**示例**

Check 该 该 natural logarithm 的 e 是 1. 
    
    
    ?Exp=exp(1);
    result: 
    2.71828 
    ?log(Exp); # Result 应该 为 1
    result: 
    1
