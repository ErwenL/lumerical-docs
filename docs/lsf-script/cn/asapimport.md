# asapimport

Imports 一个 ASAP fld 文件 into 一个 ASAP 源. This 是 equivalent 到 editing 该 属性 的 该 Import 源, 和 clicking 在 该 Import Source button. 

**语法** |  **描述**  
---|---  
asapimport( "sourcename");  |  Imports 该 fld 文件 into 该 sourcename 源. A 文件 browser 将 open 到 select 该 文件.  This 函数 does not 返回 any 数据.   
asapimport( "sourcename", "文件名");  |  Specify 该 文件 到 open.
