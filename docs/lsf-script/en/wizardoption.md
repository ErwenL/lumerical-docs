# wizardoption

Sets some options for wizard widgets and labels.  **Syntax** |  **Description**  
---|---  
wizardoption ("optionname",setting);  |  The options are 

  * "fontsize" sets the font size to any value between 8 and 40 
  * "fieldwidth" sets the width of each widget field to any value between 20 and the full width of the wizard window. 
  * "fieldheight" sets the height of each field to any value between 8 and the full height of the wizard window. 
  * "margin", sets size of the left margin to any value between 0 and full width of the wizard window.
