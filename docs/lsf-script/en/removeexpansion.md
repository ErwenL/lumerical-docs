# removeexpansion

Removes a DFT monitor from a mode expansion monitor.

| **Syntax**               | **Description**                                                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| removeexpansion("name"); | Removes the DFT monitor with the specified name from the "Monitors for expansion" list of the selected mode expansion monitor. |

**Example**

Refer to the [setexpansion](./setexpansion.md) command
