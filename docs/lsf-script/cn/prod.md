# prod

返回矩阵中元素的乘积。

**语法** | **描述**
---|---
out = prod(x); | 矩阵 x 中所有元素在所有维度上的乘积。
out = prod(x,n); | x 的元素在指定维度 n 上的乘积。

**示例**

此示例展示如何将矩阵的所有元素相乘，或仅对指定维度的元素相乘。

```powershell
?a = [1,2;3,4]; # 定义 2x2 矩阵
结果：
1 2
3 4
?prod(a); # 所有元素相乘
结果：
24
?prod(a,2); # 仅在第二维度上相乘
结果：
2
12
```
