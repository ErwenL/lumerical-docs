# ==

Performs a logical comparison. This operators can be used with complex numbers and
strings.

| **Syntax**    | **Description**                                      |
| ------------- | ---------------------------------------------------- |
| out = y == x; | Returns 1 if x and y are equal. Returns 0 otherwise. |

**Examples**

This example shows the usage of the "==" operator.

```
a=1:5;
b=1:5;
b(4)=5;
?out=a==b;
```
