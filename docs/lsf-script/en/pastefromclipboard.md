# pastefromclipboard

Pastes the contents of the system clipboard into the layout environment. Equivalent to
'Ctrl-V'.

| **Syntax**          | **Description**                    |
| ------------------- | ---------------------------------- |
| pastefromclipboard; | Paste contents of system clipboard |

**Examples**

See example in copytoclipboard function.
