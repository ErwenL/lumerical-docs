# currentfilename

返回 该 current project 文件名 和 directory. 

**语法** |  **描述**  
---|---  
out = currentfilename;  |  返回 该 current 文件名 as 一个 字符串.  If 该 current 文件名 是 not defined, 此 函数 返回 一个 empty 字符串 "".   
  
**示例**

获取 该 current 文件名. 
    
    
    ?out=currentfilename;
    C:/Users/myname/Documents/FDTD/myfile.fsp
