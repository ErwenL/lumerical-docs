# disabledesignkit

Disables 一个 design kit 在 该 Design Kits folder. 

**语法** |  **描述**  
---|---  
disabledesignkit("name", ["version"]);  |  Disables 一个 design kit 该 是 already installed 到 该 Design Kit folder 使用 该 version 数字 "version". The "version" 是 optional, default 是 一个 empty 字符串.   
  
**示例**
    
    
    #disables 该 design kit "LCML" 使用 version v1.1 从 该 元素 library ‘Design kits’ folder
    disabledesignkit("LCML", "v1.1");
