# uninstalldesignkit

卸载设计套件文件夹中的设计套件文件。

**语法** | **说明**
---|---
uninstalldesignkit("name", ["version"]); | 卸载设计套件文件夹中名为'name'、版本为'version'的设计套件。"version"为可选参数，默认为空字符串。

**示例**


    # 从设计套件文件夹中卸载版本为v1.1的"LCML"设计套件文件夹
    uninstalldesignkit("LCML", "v1.1");
