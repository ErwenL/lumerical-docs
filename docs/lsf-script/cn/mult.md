# mult

执行两个或更多矩阵的矩阵乘法。矩阵的维度必须匹配。

**语法** | **描述**
---|---
out = mult(A,B,...) | 返回矩阵 A、B、C ... 的矩阵乘法。矩阵维度必须匹配；例如，如果 A 是 MxN 矩阵且 B 是 NXP 矩阵，则 mult(A,B) 的维度为 MxP。

**示例**

求两个矩阵的矩阵乘积。

```
A = [ 1, 2; 2, 4];
B = [ 2, 5; -1, 3];
?mult(A,B);
result:
0 11
0 22
?mult(A,B,[1;2]);
result:
22
44
```
