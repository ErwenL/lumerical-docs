# customlibrary

返回 该 location (path) 的 该 Custom library. 

**语法** |  **描述**  
---|---  
out = customlibrary;  |  返回 该 directory 的 该 自定义 library.   
  
**示例**
    
    
    ?out = customlibrary;
    >C:/Users
