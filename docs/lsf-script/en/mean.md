# mean

Returns the mean value in a matrix.

| **Syntax**     | **Description**                             |
| -------------- | ------------------------------------------- |
| out = mean(a); | The mean value of the matrix a is returned. |

**Example**

Simple example showing how to find the mean value of a vector.

```
?x = linspace(0,3,4);
result: 
0 
1 
2 
3 
?mean(x);
result: 
1.5
```
