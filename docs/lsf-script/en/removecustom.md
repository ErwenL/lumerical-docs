# removecustom

Removes a folder in the Custom folder in Element Library.

**Syntax** |  **Description**  
---|---  
removecustom("foldername"); |  Removes a folder with the name specified by foldername in the Custom folder in Element Library.   
  
**Example**
    
    
    #removes the folder "new folder" in the Custom folder in Element Library
    removecustom("new folder");
