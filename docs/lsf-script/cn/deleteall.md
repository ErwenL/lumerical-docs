# deleteall

删除当前组作用域中的所有对象。

**Syntax** |  **Description**  
---|---  
deleteall; |  删除当前组作用域中的所有对象。此函数不返回任何数据。  
  
 **示例**

 此函数常用于创建组，例如[结构组](/hc/en-us/articles/360034382434-Structure-Groups)。
