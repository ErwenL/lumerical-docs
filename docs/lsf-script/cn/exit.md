# exit

退出应用程序。

**语法** | **描述**
---|---
exit; | 退出应用程序。与 exit(1) 相同；此函数不返回任何数据。
exit(option); | 退出应用程序。选项可以是

- 1：在退出前提示用户是否需要保存文件。
- 2：强制应用程序退出，不提示用户。

默认选项为 1。

**示例**

退出应用程序，并提示用户保存数据。

```powershell
exit(1);
```
