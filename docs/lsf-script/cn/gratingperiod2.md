# gratingperiod2

返回 该 grating period (i.e. 该 仿真 span) used 在 该 grating calculations. gratingperiod1 gives 该 grating period 用于 该 first 维度 (2D 和 3D). gratingperiod2 gives 该 period 的 该 2nd 维度 (3D only). See 该 grating 函数 documentation 用于 information 在 interpreting N, M, ux, uy 用于 various 监视器 orientations. 

**语法** |  **描述**  
---|---  
out = gratingperiod2( "monitorname", ...);  |  Same 参数 as grating 函数.
