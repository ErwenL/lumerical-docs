# meshgrid3dz

Create a 3D meshgrid in the z direction 

**Syntax** |  **Description**  
---|---  
out = meshgrid3dz(x,y,z);  |  The 3D version of meshgridx and meshgridy.   
  
**Examples**

See example in meshgridx.
