# getvariable

Returns the value of a workspace variable, if the value is not found it returns a
‘default’ value.

| **Syntax**                           | **Description**                                                                                    |
| ------------------------------------ | -------------------------------------------------------------------------------------------------- |
| out= getvariable (property,default); | Returns the value of a workspace variable, if the value is not found it returns a ‘default’ value. |

**Examples**

```
?getvariable("variable");
```
