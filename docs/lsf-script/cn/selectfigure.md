# selectfigure

选择图形，使其显示在屏幕上（获得焦点）。如果图形不存在，将生成警告。

**语法** | **描述**
---|---
selectfigure; | 选择最后创建的图形。此函数不返回任何数据。
selectfigure(1); | 选择图形 1。

**示例**

请参阅 exportfigure 示例。
