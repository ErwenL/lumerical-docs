# autosaveoff

This command turns off the feature to automatically save the current project before
running a simulation.

| **Syntax**   | **Description**                                                                    |
| ------------ | ---------------------------------------------------------------------------------- |
| autosaveoff; | The project will not be saved automatically before running a simulation (default). |
