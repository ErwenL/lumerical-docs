# runfinalize

此脚本命令用于完成逐步仿真。

**语法** | **描述**  
---|---  
runfinalize; | 完成逐步仿真。此命令应与 [runinitialize](runinitialize.md) 和 [runstep](runstep.md) 配合使用，通常用于联合仿真。
