# exp

计算 该 natural exponential 函数. Input 可以 为 complex. 

**语法** |  **描述**  
---|---  
out = exp(x);  |  The natural exponential 的 x.   
  
**示例**

To 计算 e, use exp(1); 
    
    
    ?exp(1);
    result: 
    2.71828 

To multiply 1+1i 通过 30 degrees 
    
    
    x=1+1i;
    ?x * exp(1i*pi/6);
    result: 
    0.366025+1.36603i
