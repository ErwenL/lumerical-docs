# ones

初始化矩阵。所有元素都设置为 1。

**语法** | **描述**
---|---
x = ones(i,j,k,....); | 初始化一个 i x j x k x .... 矩阵。

**示例**

```
?x=ones(2,2,2);
result(i,j,1):
1 1
1 1
result(i,j,2):
1 1
1 1
```
