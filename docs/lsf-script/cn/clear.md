# clear

清除所有或指定的存储工作区变量。这不会清除d-cards中存储的任何仿真数据。变量c、pi、eps0、mu0将重置为其默认值。

**语法** | **描述**
---|---
clear; | 清除所有工作区变量。此函数不返回任何数据。
clear(var1, var2, ...); | 仅清除具有指定名称的工作区变量。
