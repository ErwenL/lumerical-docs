# meshgrid3dy

在 y 方向创建 3D 网格。

**语法** | **描述**
---|---
out = meshgrid3dy(x,y,z); | meshgridx 和 meshgridy 的 3D 版本。

**示例**

参阅 meshgridx 中的示例。
