# selectall

Selects all 对象 在 该 current group scope.

**语法** |  **描述**  
---|---  
selectall; |  Selects all 对象 在 该 current group scope. This 函数 does not 返回 any 数据.  
  
**示例**

This 函数 是 often used 在 [Structure Groups](/hc/en-us/articles/360034382434-Structure-Groups).
