# zeros

Initialize a matrix. All elements are set to zero (same as [ matrix ](./matrix.md) ).

| **Syntax**             | **Description**                         |
| ---------------------- | --------------------------------------- |
| x = zeros(i,j,k,....); | Initializes an i x j x k x .... matrix. |

**Examples**

```
?x=zeros(2,2,2);
result(i,j,1):
0 0 
0 0 
result(i,j,2):
0 0 
0 0 
```
