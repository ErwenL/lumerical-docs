# &

Is 该 logical AND 函数. Imaginary components 的 x 和 y 是 ignored. 

**语法** |  **描述**  
---|---  
out = y & x;  |  If 该 real part 的 either 或 both 的 x,y 是 zero, 那么 返回 0. Otherwise 返回 1.   
y 和 x;  |  Same as &.   
  
**示例**

This example shows 该 usage 的 该 "&" 和 AND 函数. 
    
    
    ? (2) & (4);
    result:
    1
    ?(3  4) 和 (4 3);
    result: 
    0 
    ? (0+1i) & (1);
    result:
    0
