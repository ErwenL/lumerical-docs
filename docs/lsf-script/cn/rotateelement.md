# rotateelement

在原理图编辑器中旋转元素。

**语法** |  **说明**  
---|---  
rotateelement("element"); |  在原理图编辑器中旋转元素。  
  
**示例**

如果在原理图编辑器中添加了"waveguide1"，
    
    
    rotateelement("waveguide1");

将元素及其属性逆时针旋转90度。
