# clearpath

Removes all directories from the script path, except "./".

| **Syntax**              | **Description**                                        |
| ----------------------- | ------------------------------------------------------ |
| clearpath("directory"); | Remove"directory" from the script path if it is there. |

**Examples**

See addpath.
