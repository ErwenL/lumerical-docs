# autosaveoff

This 命令 turns off 该 feature 到 automatically save 该 current project before running 一个 仿真. 

**语法** |  **描述**  
---|---  
autosaveoff;  |  The project 将 not 为 saved automatically before running 一个 仿真 (default).
