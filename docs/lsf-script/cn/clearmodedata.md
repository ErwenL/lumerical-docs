# clearmodedata

**语法** | **描述**
---|---
clearmodedata; | Clears 模式 data for the selected 模式 expansion 监视器.

**示例**

Clear 模式 data stored in 模式 expansion 监视器. This will make the 文件 much smaller, which can be convenient when emailing 仿真 files. 
    select("expansion");
    clearmodedata;
