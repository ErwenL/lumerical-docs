# endl

Adds an end of line character to a string

| **Syntax**                  | **Description**                             |
| --------------------------- | ------------------------------------------- |
| out = "line1"+endl+"line2"; | Add an end of line character to the string. |

**Example**

Use an end of line character in a string.

```
?out = "line1"+endl+"line2";
line1
line2
```
