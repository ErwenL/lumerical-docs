# normpdf

计算实数值参数 x 的正态（高斯）概率密度函数（PDF）

$$ f(x)=\frac{1}{\sqrt{2\pi \sigma^{2}}}e^{-\frac{(x-\mu)^{2}}{2\sigma^{2}}} $$

其中 µ 是均值，σ 是标准差。默认情况下，μ=0 且 σ=1。

**语法** | **描述**
---|---
f = normpdf(x) | 返回实数值参数 x 的正态（高斯）概率密度函数（PDF）。默认情况下，μ=0 且 σ=1
f = normpdf(x,mu,sigma) | 返回实数值参数 x 的正态（高斯）概率密度函数（PDF）。µ 是均值，σ 是标准差。
