# killwizard

This closes the wizard window. It should only be called after a wizard window has been
created with the newwizard command.

| **Syntax**  | **Description**                |
| ----------- | ------------------------------ |
| killwizard; | This closes the wizard window. |

**Examples**

See the newwizard page for an example.
