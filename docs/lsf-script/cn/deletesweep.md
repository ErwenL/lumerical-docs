# deletesweep

Deletes 一个 specified 参数 sweep, optimization, 或 Monte Carlo 分析 task.

**语法** |  **描述**  
---|---  
deletesweep("name"); |  Deletes 该 sweep, optimization, 或 Monte Carlo 分析 task 使用 该 specified name.  
  
**示例**

This example shows 如何 到 delete 一个 task named "thickness_sweep". Please download 该 example 文件 从 该 [ Parameter sweeps ](/hc/en-us/articles/360034922873-Parameter-sweeps) page Associate files.
    
    
    deletesweep("thickness_sweep");
