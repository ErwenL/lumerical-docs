# meshgrid3dy

Create a 3D meshgrid in the y direction

| **Syntax**                | **Description**                            |
| ------------------------- | ------------------------------------------ |
| out = meshgrid3dy(x,y,z); | The 3D version of meshgridx and meshgridy. |
