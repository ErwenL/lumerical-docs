# runwizard

运行向导并返回一个值，指示按下了哪个按钮。

**语法** |  **描述**  
---|---  
out = runwizard;  |  返回 0、+1 或 -1。0 表示用户按下了取消按钮，1 表示用户按下了第一个按钮（通过调用 newwizardpage 创建），-1 表示用户按下了第二个按钮（通过调用 newwizardpage 创建）。  
  
**示例**

请参阅 newwizard 页面获取示例。
