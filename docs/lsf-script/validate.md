# validate

Updates the results for the specified analyzer.

**Syntax** |  **Description**  
---|---  
validate("analyzer"); |  Updates the results for the specified "analyzer". If the name is not provided, the selected analyzer will be updated.  
  
**Example**
    
    
    addelement("Optical Network Analyzer");  
    validate("ONA_1");
