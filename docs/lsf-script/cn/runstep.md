# runstep

此脚本命令运行单个仿真步骤。

**语法** | **描述**  
---|---  
runstep; | 运行单个仿真步骤。此命令应与 [runinitialize](runinitialize.md) 和 [runfinalize](runfinalize.md) 配合使用，通常用于联合仿真。
