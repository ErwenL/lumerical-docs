# removecustom

删除元件库中 Custom 文件夹中的文件夹。

**语法** | **描述**
---|---
removecustom("foldername"); | 删除元件库中 Custom 文件夹中名称为 foldername 的文件夹。

**示例**

```lsf
# 删除元件库中 Custom 文件夹中的 "new folder" 文件夹
removecustom("new folder");
```
