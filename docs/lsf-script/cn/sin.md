# sin

计算三角正弦函数。角度单位以弧度表示。该函数定义用于复角。复数的相位在-π到π之间求值。

**语法** | **描述**
---|---
out = sin(x); | 返回x的复正弦值。

**示例**

计算sin(π/4 + i)。

```
theta=pi/4+1i;
?sin(theta);
结果：
1.09112+0.830993i
```
