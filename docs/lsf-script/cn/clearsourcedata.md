# clearsourcedata

**语法** | **描述**
---|---
clearsourcedata; | Clears 光源 data for the selected 光源.

**示例**

Clear 光源 data from an imported 光源. This will make the 文件 much smaller, which can be convenient when emailing 仿真 files. 
    select("source3");
    clearsourcedata;
