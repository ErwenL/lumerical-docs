# getsetting

Returns the value of a user defined setting. This command can be used with [ setsetting ](/hc/en-us/articles/360034927973-setsetting) . 

**Syntax** |  **Description**  
---|---  
getsetting("name");  |  Gets the value of a user defined setting.   
  
**Parameter** |  **Type** |  **Description**  
---|---|---  
name  |  string  |  name of the setting.   
  
**Example**

Please visit the page [ setsetting ](/hc/en-us/articles/360034927973-setsetting) .
