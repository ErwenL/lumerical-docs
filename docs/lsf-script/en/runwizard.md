# runwizard

Runs the wizard and returns a value indicating which button was pressed.

| **Syntax**       | **Description**                                                                                                                                                                                                               |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| out = runwizard; | Returns either 0, +1 or -1. 0 means the user pressed Cancel, 1 means the user pressed the first button (created by calling newwizardpage) and -1 means the user pressed the second button (created by calling newwizardpage). |
